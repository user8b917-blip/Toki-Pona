Baralho de Toki Pona

Este baralho reúne palavras essenciais de Toki Pona com definições curtas para facilitar memorização e leitura rápida.

O foco do deck é aprender o significado das palavras, entender partículas, montar frases, reconhecer contexto e modificadores.

O baralho inclui:

Pronomes, verbos, substantivos, cores, números, emoções e partículas gramaticais.

As definições foram simplificadas, então algumas palavras possuem outros usos dependendo do contexto.
