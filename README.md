Baralho de Toki Pona

Este baralho reúne palavras essenciais de Toki Pona com definições curtas em português para facilitar memorização e leitura rápida.

O foco do deck é:

aprender o significado central das palavras.
entender partículas importantes.
montar frases.
reconhecer contexto e modificadores.

O baralho inclui:

pronomes.
verbos
substantivos.
cores.
números.
emoções.
partículas gramaticais.


As definições foram simplificadas, então algumas palavras possuem outros usos dependendo do contexto.
